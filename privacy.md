# Privacy Policy for auto_tiktok

**Last updated: April 10, 2026**

## 1. Introduction

This Privacy Policy describes how auto_tiktok (the "Service") handles information in connection with its use. auto_tiktok is a personal tool that operates locally on the user's own device. The developer does not operate any server that collects or stores user        
information.

## 2. Information We Collect

The Service does not collect, store, or transmit any personal information to servers operated by the developer. All data handled by the Service remains on the user's local device.

## 3. TikTok API Access

To publish videos to TikTok, the Service uses the TikTok Content Posting API with OAuth 2.0 authentication. The following data is handled strictly for the purpose of authorized API communication with TikTok:

- **Access Tokens and Refresh Tokens**: Stored locally on the user's device in protected configuration files. These tokens are never transmitted to any party other than TikTok's official API endpoints.
- **User Profile Data**: Only the minimum information required to identify the target TikTok account (such as `open_id` and username) is stored locally.
- **Video Files**: Videos are uploaded directly from the user's device to TikTok servers via the official API. Video content is not stored, processed, or transmitted by any intermediary operated by the developer.
- **Publishing Metadata**: Captions, titles, and publishing status are sent directly to TikTok via the API.

## 4. Data Storage and Retention

All data handled by the Service is stored locally on the user's device. The user has full control over this data and can delete it at any time by removing the application files. The developer does not retain any user data on any external server.

## 5. Third-Party Services

The Service communicates only with official TikTok API endpoints. No data is shared with any third party other than TikTok.

## 6. Security

The user is responsible for securing their local device and the configuration files that contain API credentials. The developer recommends storing credential files in a directory that is not shared or synchronized with untrusted services.

## 7. Children's Privacy

The Service is not intended for use by individuals under the age of 13. In accordance with TikTok's own minimum age requirement, users must be at least 13 years old (or the minimum age required by applicable law in their jurisdiction) to use the Service.

## 8. User Rights

Because the Service does not collect or store personal information on any server operated by the developer, there is no remote data to access, modify, or delete. Users retain full control over all locally stored data.

## 9. Changes to This Privacy Policy

This Privacy Policy may be updated from time to time. Any changes will be reflected by updating the "Last updated" date at the top of this page. Your continued use of the Service after such changes constitutes acceptance of the revised Privacy Policy.
